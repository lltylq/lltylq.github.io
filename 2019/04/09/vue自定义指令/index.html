vue和angular一样都自己带了自定义指令的方法，vue的自定义指令分为全局的和局部的接下来我说一下vue指令的用法。

1、定义全局的自定义指令

全局中自定义指令：在main.js的头部自动以指令 ，下面就注册了一个元素调用 v-focus,就能在元素插入dom的时候，自动获取焦点 。

2、定义局部的指令

​ 定义布局的指令，需要在我们的new Vue实例中将我们的directives相关的逻辑写入。

3、自定义指令中的钩子函数

  • bind：只调用一次，用这个钩子函数可以定义一个绑定时执行初始化的动作（是在渲染dom元素之前的操作）
  • inserted：被绑定元素插入父节点时调用
  • update：所在组件的 VNode 更新时调用，但是可能发生在其孩子的 VNode 更新之前。指令的值可能发生了改变也可能没有。
  • componentUpdated：所在组件的 VNode 及其孩子的 VNode 全部更新时调用。

3.1钩子函数的参数

  • el：所绑定元素，可以直接操作DOM元素。
  • binding：一个独享，包含以下属性
    • name：指令名
    • oldValue：指令绑定的前一个值，仅在update和componentUpdated钩子中可用。无论值是否改变都可用。
    • ：指令的绑定值
    • arg：传给指令的参数。例如v-focus:foo，arg 的值是”foo”
    • oldValue：指令绑定的前一个值，仅在update和componentUpdated钩子中可用。无论值是否改变都可用。
    • modifiers：一个包含修饰符的对象。例如：v-my-directive.foo.bar, 修饰符对象 modifiers 的值是{ foo: true, bar: true }。
    • vnode：Vue 编译生成的虚拟节点
    • oldVnode：上一个虚拟节点，仅在update和componentUpdated 钩子中可用。
